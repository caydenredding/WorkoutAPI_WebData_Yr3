# Which Framework to use
The options were Django or FastAPI
Django requires more work to setup and slower to work with (even if it has more built in features etc.)
FastAPI is lightweight which means for the purposes of this project it is quicker to set-up. Also has better AI intergration which allows for more relevant novel features to be included later.

FastAPI is also allows rapid development of RESTful endpoints - which allows more features to be developed for the user.
